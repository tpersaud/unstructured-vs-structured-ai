# Kata Run Prompts

Max 5 prompts per run.

## Unstructured Context

### 1 Plan

Plan implementation from task spec.

### 2 Ask

Confirm scope.

### 3 Code

Implement.

### 4 Validate

Review correctness.

### 5 Refactor

Fix issues or clean up.

---

## Structured Context

Same steps, but:

- must use structured-context artifacts
- must follow rules/workflows
- must align with structured context seed
