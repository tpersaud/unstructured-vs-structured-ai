# Kata Run Prompts

Max 5 prompts per run.

## Prompt-only

### 1 Plan

Plan implementation from task spec.

### 2 Ask

Confirm scope.

### 3 Code

Implement.

### 4 Validate

Review correctness.

### 5 Refactor

Fix issues or clean up.

## Lattice

Same steps, but:

- must use lattice artifacts
- must follow rules/workflows
- must align with structured context
